🤖 Installation Android
1

📱 Via Chrome (Recommandé)

La méthode la plus simple — Chrome installe GATE 75 comme une vraie app Android.

  • Ouvrez Chrome sur votre Android
  • Allez sur gate75.netlify.app
  • Attendez 10 secondes — une bannière "Ajouter à l'écran d'accueil" apparaît automatiquement
  • Appuyez sur Installer ou Ajouter
  • GATE 75 apparaît sur votre écran d'accueil comme une vraie application !
💡 Si la bannière n'apparaît pas : Menu Chrome (⋮) → "Ajouter à l'écran d'accueil"
2

📦 Via PWABuilder (APK signé)

Génère un vrai fichier APK installable sur n'importe quel Android sans passer par le Play Store.

  • Allez sur pwabuilder.com
  • Entrez l'URL : https://gate75.netlify.app
  • Cliquez "Start" — il analyse votre PWA
  • Cliquez "Package for stores"
  • Choisissez "Android" → "Generate Package"
  • Téléchargez le fichier .apk
  • Envoyez-le sur votre téléphone et installez-le
⚠️ Activez "Sources inconnues" dans Paramètres → Sécurité avant d'installer l'APK
🔗 Ouvrir PWABuilder
3

🛠️ Via Bubblewrap (APK avancé)

Outil officiel Google pour convertir une PWA en APK TWA (Trusted Web Activity) — identique au Play Store.

  • Installez Node.js sur votre ordinateur
  • Installez Bubblewrap : npm i -g @bubblewrap/cli
  • Initialisez : bubblewrap init --manifest https://gate75.netlify.app/manifest.json
  • Compilez : bubblewrap build
  • Récupérez le fichier app-release-signed.apk
📌 Nécessite Java JDK et Android SDK — recommandé pour la migration Play Store

🧭 Via Safari (Seule méthode iOS)

Sur iPhone, seul Safari permet d'installer des PWA sur l'écran d'accueil.

  • Ouvrez Safari (pas Chrome, pas Firefox)
  • Allez sur gate75.netlify.app
  • Appuyez sur le bouton Partager (icône 📤 en bas)
  • Faites défiler et appuyez sur "Sur l'écran d'accueil"
  • Appuyez sur "Ajouter" en haut à droite
  • GATE 75 est installé sur votre iPhone !
📌 iOS 16.4+ : les PWA ont accès aux notifications push. iOS 17+ : expérience optimale.
